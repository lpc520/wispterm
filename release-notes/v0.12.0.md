# Phantty v0.12.0

This release focuses on making Phantty a better place to inspect files without
leaving the terminal.

## Highlights

- Added a right-side Markdown/text preview panel for `.md` and `.txt` files.
- Added Ctrl-click preview support from terminal output and from File Explorer.
- Added WSL File Explorer support through `wsl.exe`.
- Added SSH Markdown/text preview for sessions launched from Phantty SSH profiles.
- Added resizable and scrollable preview panel layout beside the File Explorer.
- Added URL underline feedback so Ctrl-clickable URLs are easier to discover.

## Fixes and polish

- Fixed SSH preview paths that start with `~` by keeping `$HOME` expandable in
  remote shell commands.
- Improved Markdown rendering for headings, lists, blockquotes, code blocks,
  inline code, links, and horizontal rules.
- Kept terminal layout from overlapping with the File Explorer and preview
  panel when right-side panels are visible.

## Known limitation

SSH previews currently require Phantty's SSH profile metadata. A session created
by manually typing `ssh user@host` inside a local shell is still treated as a
local shell, so remote preview is not available for that path yet.
