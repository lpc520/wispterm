# Phantty v0.16.0

This release focuses on remote access usability, especially on iPhone-sized
screens. The remote web console now uses a terminal-first mobile layout with a
compact utility keyboard and a more reliable selected-surface input path.

## Highlights

- Redesigned the mobile remote console around four compact regions: top bar,
  surface strip, selected terminal, and bottom utility keyboard.
- The selected remote surface now fits the mobile viewport instead of keeping
  the remote split grid geometry on small screens. Desktop remote rendering
  still preserves the remote grid dimensions.
- Added a hidden mobile text input bridge so tapping the terminal or Type key
  can use the native mobile keyboard while still sending input to the selected
  surface.
- Improved IME handling for the mobile input bridge, including composition
  guards and exact-once committed text dispatch.
- Tightened the virtual keyboard input helpers and fixed sticky Ctrl/Alt
  modifiers so they clear after the next text or special key.

## Other changes

- Added text-file preview support alongside Markdown preview.
- Added mouse multi-click selection behavior.
- Changed split focus shortcuts to `Ctrl+Shift` plus arrow keys and updated
  the website shortcut documentation.
- Added the GitHub Pages website and Chinese documentation page.
- Fixed a PageList resize race during panel dragging.
