# Phantty v0.19.0

## Highlights

- AI Chat can run as an **agent**: OpenAI-compatible tool calling with a
  multi-turn loop over terminal observation, local PowerShell execution, and
  commands injected into **already-open SSH** sessions.
- **Permission control**: `Ask` waits for an in-chat approval card
  (Enter/Y approve, Esc/N deny); `Full` runs tools without prompting.
- Session launcher shows **AI** with **Agent** or **Chat** based on the saved
  profile.

## AI Agent

- Tools: `terminal_list`, `terminal_snapshot`, `powershell_exec`,
  `ssh_session_exec` (snapshots refresh on each call).
- Config keys: `ai-agent-enabled`, `ai-agent-permission`, `ai-agent-command-timeout-ms`,
  `ai-agent-output-limit`.
- Replays `reasoning_content` on follow-up requests for thinking-style models.
- Sanitizes invalid UTF-8 in outbound JSON so providers do not reject payloads.
- Local shell runs honor the configured timeout and kill stuck processes.

## UI

- Header shows **Mode: Agent / Chat** and a clickable **Ask / Full** permission chip.
- Dedicated approval card at the bottom when a tool needs confirmation.

## Fixes

- Widened header slots so mode and permission labels are not clipped.
