# Phantty v0.19.3

## Highlights

- Added terminal agent status detection and badges for AI-assisted terminal
  workflows, including Codex and Claude Code.
- Added REPL-aware agent controls so AI tooling can send input to existing R,
  Python, Codex, Claude Code, or plain terminal sessions.
- Streamlined AI Agent session launch so the default profile opens directly in
  agent mode from the session launcher.
- Synced local AI chat and terminal agent metadata through the remote web
  console so browser clients can see and control the same session state.

## Fixes

- Fixed stale or incorrect Codex status detection around approvals, sandbox
  retry prompts, running state, completion, and halted output.
- Added Claude Code approval and running-state detection for edit prompts and
  tool activity such as `Update(...)`, `Read(...)`, and `Bash(...)`.
- Fixed scrollback selection copying.
- Fixed SSH password prompt timing so saved profiles still prompt reliably
  after reconnect delays.
