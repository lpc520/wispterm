# Phantty v0.23.1

## Improvements

- Embedded the default Agent system prompt from `src/prompt.md`, with Windows PowerShell, Phantty terminal tool, SSH/WSL, Codex/Claude Code, and `uv` guidance.
- Increased the AI profile system prompt field capacity so custom prompts can hold the larger default guidance.
- Installed the tracked `plugins/` root with the application package so future plugin and skill assets have a stable runtime location.
- Added a `phantty-windows-portable-no-webview-vX.Y.Z.zip` release asset built with embedded WebView2 disabled.
