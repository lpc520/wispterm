# Phantty v0.24.0

## New

- Added startup update checks against GitHub Releases. Phantty now shows a clickable prompt when a newer release is available, and the command center includes `Check for Updates` and `Open Latest Release`.
- Added the `auto-update-check` config option so startup update checks can be disabled.
- Added drag-and-drop reordering for tabs in the left tab sidebar.

## Remote

- Added remote AI Agent open requests so Phantty Remote can ask the desktop app to open the default Agent session for supported Weixin/iLink prompts.
- Improved remote AI Agent open edge-case handling, including disconnect and unavailable-profile cases.
- Fixed initial remote snapshot replay so reconnecting web clients receive the current terminal state more reliably.
