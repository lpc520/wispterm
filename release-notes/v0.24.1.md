# Phantty v0.24.1

## Fixes

- Normalized Windows clipboard newlines before paste encoding so pasted text
  behaves consistently in bracketed and non-bracketed paste modes.
- Added a clickable SSH drag-and-drop upload prompt when Phantty has to fall
  back to a fresh `ssh.exe pwd` helper because the remote shell did not report
  OSC 7 current-directory state.

## Documentation

- Added SSH current-directory setup snippets for Bash, Zsh, and Fish so drag
  and drop uploads can target the active remote shell directory instead of the
  SSH login directory.
