# Phantty v0.24.3

## Changes

- Added `--config` / `--config-path` for selecting the main config file at startup.
- Added portable config discovery via `phantty.conf` next to `phantty.exe`, ahead of `%APPDATA%\phantty\config`.
- Fixed CLI command output paths so release builds can print help, version, theme, font, and config-path output to stdout.
