# Phantty v0.26.0

## New

- Added Ctrl+Shift-click downloads for SSH terminal file paths, saving remote
  files to the local Downloads folder.
- Added right-corner transfer toasts for downloads and uploads so background
  SSH transfers are visible even when File Explorer is closed.

## Improvements

- File Explorer, Markdown preview, and browser panels are now bound to the tab
  that opened them instead of being preserved across tab switches.
- Markdown/text previews now load in the background instead of blocking the UI.
- New Session PowerShell entries now honor the configured shell, including
  PowerShell 7 paths under `Program Files`.
- Ctrl-hover underline feedback now covers preview/download paths, including
  plain remote filenames such as `sample.h5ad`.
- SSH file uploads and downloads now run as queued background jobs instead of
  blocking the UI thread.

## Fixes

- Preserved SSH drag-and-drop upload paste semantics by waiting until upload
  success before pasting the remote path.
- Fixed multi-file SSH drops so subsequent files are queued instead of ignored.
