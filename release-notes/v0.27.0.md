# Phantty v0.27.0

## New

- Added image preview panel support for files opened from File Explorer and
  terminal path previews.
- Added pan interaction for zoomed image previews.

## Fixes

- Fixed Kitty graphics image rendering so image cells render with the correct
  source orientation.
- Fixed WSL `imgcat` rendering by handling Kitty graphics chunk data more
  robustly before it reaches the renderer.
- Fixed a File Explorer reorder test expectation so tab/path ordering coverage
  matches the current behavior.
