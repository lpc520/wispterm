# Phantty v0.29.0

## New

- Added CSV and TSV file previews with table-style rendering.
- Added hover detail popups for truncated table preview cells.

## Improvements

- Increased the maximum width for the left sidebar, right sidebar, preview
  panel, and browser panel so wide file previews have more room.
- Improved mouse word selection by using terminal-style delimiters:
  ``\ :;~`!@#$%^&*()=+|[]{}'",<>?``.

## Fixed

- Fixed application cursor mode so full-screen tools such as `less -S` handle
  arrow keys correctly.
- Fixed selection highlight opacity on light and dark backgrounds so partial
  selections are visually distinguishable from full-line selections.
- Fixed IME caret placement for visual-cursor terminal apps such as Claude Code.
