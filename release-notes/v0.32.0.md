# Phantty v0.32.0

## Added

- **AI Chat: OpenAI Responses API support.** AI profiles now have a `Protocol`
  field — `chat_completions` (default) or `responses`. Set it to `responses`
  with a base URL like `https://api.openai.com/v1` (or a full `/responses`
  endpoint) to talk to OpenAI Responses API providers; tool calls, reasoning,
  streaming, and token usage are handled for both protocols.
- **`url-open-mode` config.** Choose where `Ctrl`-clicked web URLs open:
  `embedded` (default — the right-side WebView2 panel when available) or
  `system-browser` (always the Windows default browser). SSH loopback URLs are
  tunneled in both modes.

## Fixed

- Corrected the rendering order on DPI changes and window resizes, fixing
  transient artifacts when moving between monitors or resizing.

## Internal

- Began decoupling the renderer's GPU layer behind a backend interface
  (`src/renderer/gpu/`), with the terminal grid (`cell_renderer`) and the
  titlebar/shared UI rendering now routed through it. This is groundwork toward
  native macOS and Linux ports; there is no user-visible change on Windows.
