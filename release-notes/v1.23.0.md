# WispTerm v1.23.0

> This release adds **SSH key and credential authentication** for saved
> connection profiles, makes **Settings changes apply instantly**, and brings
> **remote access to macOS** (early). It also fixes macOS config/font-size
> hot-reload, overlays that overflowed off-screen on short windows,
> double/triple/quad-click selection on macOS, copying soft-wrapped lines, and
> tmux session-restore glitches.

## Added

- **SSH key and credential authentication.** Saved SSH profiles can now
  authenticate with a private key (identity file) or delegate to your system
  credentials (OpenSSH config, default keys, ssh-agent), in addition to a saved
  password. The Session Launcher gains **Auth method** and **Identity file**
  fields, and the Copilot's `ssh_profile_save` tool accepts the same options.
  Existing password profiles keep working unchanged. (#246)
- **Remote access on macOS (early).** Remote access — the opt-in relay
  connection enabled via `remote-enabled` — now has a native WebSocket transport
  on macOS, where it previously had no backend (enabling it silently failed and
  looped on reconnect). Off by default; treat as experimental until verified
  end-to-end against a relay. (#231)

## Changed

- **Settings apply instantly.** Changes made on the Settings page now take
  effect on the running terminal immediately instead of after a short delay —
  theme, font, and other options update right away. (#243)

## Fixed

- **macOS config and font-size hot-reload.** On macOS, changing settings
  (especially font size) or editing the config file now takes effect immediately
  instead of requiring a restart, matching Windows. (#242)
- **Overlays scroll on short windows.** On short windows, the Settings, Command
  Center, Session launcher, Jupyter picker, and Skill Center panels overflowed
  off-screen, hiding lower rows (including the Close button). They now clamp to
  the window and scroll, with mouse-wheel support and a scrollbar, so every
  option stays reachable. (#233)
- **Word/line/paragraph selection on macOS.** Double-, triple-, and
  quad-clicking now selects a word, line, and paragraph on macOS (previously a
  no-op). Triple-click now selects the whole line on all platforms, matching
  macOS Terminal and iTerm2. (#240)
- **Copying soft-wrapped lines.** Selecting and copying a long line that visually
  wraps across rows (e.g. a long path) now copies back as a single line instead
  of being split at each wrap point; real line breaks are still preserved. (#237)
- **tmux session restore.** Restoring or reconnecting a `tmux -CC` control-mode
  session no longer leaks stray internal text or title fragments into panes;
  full-screen TUIs like `top` restore on the correct screen with the right
  cursor position, and an established session reattaches instead of being
  recreated. (#239)

## Docs

- **Windows FAQ.** New entries explain why "select-to-translate" tools can
  interrupt a running program with `^C` (they synthesize Ctrl+C, which is SIGINT
  in a terminal) and how to fix it, plus a reminder to fully extract the compat
  zip so `conpty.dll` / `OpenConsole.exe` stay beside `wispterm.exe`. (#241)

## Known limitations

- **Linux is experimental.** The Linux AppImage is published for community
  testing and is not yet considered stable.
- **macOS remote access is early.** The macOS relay transport is new and only
  verified on-device against a live relay; report issues if reconnect loops or
  messages don't arrive.

# WispTerm v1.23.0（中文）

> 本版本为已保存的连接配置新增 **SSH 密钥与凭据认证**,让 **设置即时生效**,并为
> **macOS 带来远程访问支持(早期)**。同时修复了 macOS 配置/字号热重载、短窗口下
> 浮层超出屏幕、macOS 双击/三击/四击选择、软换行长行的复制,以及 tmux 会话恢复的
> 若干问题。

## 新增

- **SSH 密钥与凭据认证。** 已保存的 SSH 配置现在除保存密码外,还可以使用私钥
  (identity 文件)认证,或委托给系统凭据(OpenSSH 配置、默认密钥、ssh-agent)。
  Session Launcher 新增「认证方式」和「Identity 文件」字段,Copilot 的
  `ssh_profile_save` 工具也支持同样的选项。已有的密码配置保持不变。（#246）
- **macOS 远程访问(早期)。** 远程访问——通过 `remote-enabled` 开启的中继连接——
  现在在 macOS 上有了原生 WebSocket 传输(此前 macOS 没有后端,开启后会静默失败并
  不断重连)。默认关闭;在与中继服务器端到端验证之前视为实验性功能。（#231）

## 调整

- **设置即时生效。** 在设置页所做的更改现在会立即作用到正在运行的终端,而不再有短暂
  延迟——主题、字体等选项会即时更新。（#243）

## 修复

- **macOS 配置与字号热重载。** 在 macOS 上,更改设置(尤其是字号)或编辑配置文件现在
  会立即生效,无需重启,与 Windows 行为一致。（#242）
- **短窗口下浮层可滚动。** 在较矮的窗口下,设置、Command Center、Session launcher、
  Jupyter 选择器和 Skill Center 等浮层会超出屏幕,导致下方的行(包括关闭按钮)被遮挡。
  现在这些浮层会被夹在窗口内并可滚动,支持鼠标滚轮和滚动条,确保每个选项都点得到。
  （#233）
- **macOS 词/行/段落选择。** 双击、三击、四击现在在 macOS 上分别选择单词、整行和段落
  (此前无效)。三击现在在所有平台都选择整行,与 macOS 终端和 iTerm2 一致。（#240）
- **复制软换行的长行。** 选中并复制一条在视觉上跨多行换行的长行(例如长路径),现在会
  复制为单行,而不再在每个换行点被拆开;真实的换行仍会保留。（#237）
- **tmux 会话恢复。** 恢复或重连 `tmux -CC` 控制模式会话不再把内部杂项文本或标题片段
  泄漏到窗格里;像 `top` 这样的全屏 TUI 会在正确的屏幕上以正确的光标位置恢复,已建立的
  会话会重新接入而不是被重新创建。（#239）

## 文档

- **Windows FAQ。** 新增条目说明为什么「划词翻译」类工具会用 `^C` 打断正在运行的程序
  (它们会合成 Ctrl+C,在终端中即 SIGINT)以及如何解决,并提醒完整解压 compat zip 以
  保证 `conpty.dll` / `OpenConsole.exe` 与 `wispterm.exe` 在同一目录。（#241）

## 已知限制

- **Linux 仍为实验性质。** Linux AppImage 用于社区测试,尚不视为稳定版本。
- **macOS 远程访问为早期功能。** macOS 中继传输是全新功能,仅在设备上针对真实中继做过
  验证;如遇重连循环或消息收不到,请反馈。
