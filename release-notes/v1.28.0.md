# WispTerm v1.28.0

> This release makes the Copilot sidebar persistent and resumable, lets sidebar
> Copilot conversations export Markdown, adds first-party Agent tool toggles, and
> fixes SSH profile management plus PDF preview panning.

## Added

- **Persistent Copilot sidebar conversations (#283).** Sidebar Copilot chats are
  now saved into the existing agent-history store, restored in place when tabs or
  windows are restored, and kept separate from regular AI Chat tabs.
- **Copilot conversation picker (#283).** `/resume`, **Load Copilot Conversation**
  in the command center, and `Ctrl/Cmd+Shift+R` open a sidebar conversation
  picker with newest-first rows, `+ New`, delete, and de-dup that switches to an
  already-open live tab instead of opening the same conversation twice.
- **First-party Agent tool toggles (#280).** Built-in Agent tools now appear in
  Skill Center and can be disabled/enabled separately from imported executable
  tools. Disabled first-party tools are hidden from new request schemas and stale
  runtime calls are rejected.
- **macOS UI end-to-end test harness (#277).** The repository now has a real
  macOS UI automation harness for keyboard, menu, clipboard, and agent-control
  smoke tests.

## Fixed

- **Copilot sidebar Markdown export (#286).** `/export` and command-driven
  Markdown export now target the sidebar Copilot session that invoked the export
  instead of requiring a dedicated AI Chat tab.
- **SSH server list management (#281).** Saved/imported SSH profiles are no
  longer capped at 16 entries, the server list has a visible search field and
  bounded scrolling, and profile deletion supports multi-select with a
  confirmation row.
- **PDF preview drag pan (#282).** Zoomed PDF previews can now be dragged to pan,
  matching image preview behavior.
- **Compact Copilot status indicator (#284).** Narrow Copilot sidebars now show a
  compact colored status dot instead of a wide status label, while full AI Chat
  tabs keep the readable status text.

## Known limitations

- **Linux is experimental.** The Linux AppImage is published for community
  testing and is not yet considered stable.
- **macOS remote access remains early.** The macOS relay transport is available
  but still needs more end-to-end field testing.

# WispTerm v1.28.0（中文）

> 本版本让 Copilot 侧边栏会话可持久化、可恢复，并支持侧边栏 Copilot 导出
> Markdown；同时增加内置 Agent 工具开关，修复 SSH profile 管理与 PDF 预览拖拽平移。

## 新增

- **Copilot 侧边栏会话持久化（#283）。** 侧边栏 Copilot 聊天现在会保存到现有
  agent-history 存储中，恢复标签页/窗口时会原地恢复，并与普通 AI Chat tab 历史保持分离。
- **Copilot 会话选择器（#283）。** `/resume`、命令中心的 **Load Copilot Conversation**
  以及 `Ctrl/Cmd+Shift+R` 会打开侧边栏会话选择器，按最新更新时间排序，支持 `+ New`、删除，
  并会在同一会话已打开时切换到对应 tab，避免重复打开。
- **内置 Agent 工具开关（#280）。** Skill Center 现在会显示内置 Agent 工具，并允许像管理
  导入的可执行工具一样单独启用/停用。停用的内置工具不会进入新请求的 schema，旧的运行时
  调用也会被拒绝。
- **macOS UI 端到端测试 harness（#277）。** 仓库新增真实 macOS UI 自动化测试框架，用于
  键盘、菜单、剪贴板和 agent-control 冒烟测试。

## 修复

- **Copilot 侧边栏 Markdown 导出（#286）。** `/export` 和命令触发的 Markdown 导出现在会
  指向发起导出的侧边栏 Copilot 会话，不再要求先打开独立 AI Chat tab。
- **SSH server 列表管理（#281）。** 保存/导入的 SSH profile 不再限制为 16 条；server 列表
  增加可见搜索框和固定高度滚动视口，并支持多选删除与确认行。
- **PDF 预览拖拽平移（#282）。** 放大后的 PDF 预览现在可以像图片预览一样拖拽平移。
- **紧凑 Copilot 状态指示（#284）。** 狭窄的 Copilot 侧边栏现在用彩色状态圆点替代宽状态
  文本；完整 AI Chat tab 仍保留可读状态文字。

## 已知限制

- **Linux 仍为实验性质。** Linux AppImage 用于社区测试，尚不视为稳定版本。
- **macOS 远程访问仍为早期功能。** macOS 中继传输已经可用，但仍需要更多端到端实际验证。
