# WispTerm v1.4.0

> AI Chat gets easier to steer: conversations can now title themselves after the
> first turn, and double-Esc opens a rewind picker so you can jump back to an
> earlier exchange without rebuilding context by hand.

## Added

- **AI Chat auto-title.** New conversations can generate a compact title after
  the first user turn, while preserving restored or manually named sessions.
- **Double-Esc rewind picker.** Press Esc twice in AI Chat to choose an earlier
  message point and restore the composer from there.
- **Settings toggle for WeChat direct.** The built-in Settings page now includes
  `WeChat direct  on/off`, writing `weixin-direct-enabled` without opening the
  raw config file.

## Changed

- **WeChat direct and Remote Weixin bindings coexist.** The desktop Zig direct
  binding is independent from the Remote server's Weixin bridge binding; enabling
  or unbinding one no longer disables or clears the other.

## Fixed

- **macOS text input.** Option-modified generated text is preserved, and raw key
  events are no longer forwarded while IME composition is active.
- **macOS app bundle contents.** Default plugins and skills are bundled into the
  `.app`, so packaged builds have the same built-in capabilities as development
  builds.
- **AI Chat selection behavior.** Clearing a selection no longer primes the
  double-Esc rewind gesture.

# WispTerm v1.4.0（中文）

> AI Chat 更好掌控了：首轮对话后可自动生成标题，双击 Esc 可打开 rewind 选择器，
> 回到之前的对话节点，而不用手动重建上下文。

## 新增

- **AI Chat 自动标题。** 新对话可在首轮用户消息后生成简短标题，同时保留恢复会话或手动
  命名会话的标题。
- **双 Esc rewind 选择器。** 在 AI Chat 中连续按两次 Esc，可选择更早的消息节点并从那里
  恢复输入。
- **Settings 里的 WeChat direct 开关。** 内置 Settings 页面新增
  `WeChat direct  on/off`，可直接写入 `weixin-direct-enabled`，无需打开原始配置文件。

## 变更

- **WeChat direct 与 Remote Weixin 绑定可共存。** 桌面 Zig 本地直连绑定与 Remote
  server 的 Weixin bridge 绑定彼此独立；启用或解绑其中一个，不会禁用或清除另一个。

## 修复

- **macOS 文本输入。** 保留 Option 组合键生成的文本，并在 IME 组合输入期间不再转发原始
  key event。
- **macOS app bundle 内容。** 默认 plugins 与 skills 会打包进 `.app`，打包构建与开发
  构建拥有一致的内置能力。
- **AI Chat 选区行为。** 清除选区不再触发双 Esc rewind 手势的预备状态。
