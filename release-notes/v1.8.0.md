# WispTerm v1.8.0

> This release expands the AI chat and AI History workflows, adds safer close
> confirmation for running programs, and fixes several Windows rendering and
> agent-execution regressions.

## Added

- **Vision image paste for AI chat.** `Ctrl+Shift+V` can paste clipboard images
  into the AI composer for vision-capable models, with visible attachment
  placeholders and a top-right paste toast.
- **Dropped file paths in AI chat.** Dropping a local file over a chat composer
  inserts its path at the cursor when the drop targets the active chat surface.
- **Reasonix AI History support.** AI History can now discover and display
  Reasonix history alongside existing providers.
- **Panel management controls.** Right-side panels now expose close buttons, and
  `Alt`-drag can swap panels within a tab.
- **Running-program close confirmation.** Closing tabs, splits, or the window
  now asks for confirmation when a terminal still has a running program, with a
  new `confirm-close-running-program` config toggle.
- **Agent terminal recovery controls.** Agent terminal execution can send control
  keys for recovery and blocks new command injection while a previous command is
  still running.

## Fixed

- **AI API request errors are visible.** Failures from AI API requests now
  surface to the user instead of being hidden behind a silent failure path.
- **AI chat image composer rendering.** Pasted image references are shown above
  the composer, placeholders are inserted consistently, and approval cards scale
  with the UI font size.
- **Agent exec false failures.** Completion sentinel matching now uses the
  status marker instead of echoed command text, reducing false timeout/failure
  reports.
- **Windows rendering glitches.** Fixed fullscreen black screens, multi-monitor
  icon ghosting, and resize stutter.
- **Preview path punctuation.** Fullwidth punctuation in preview paths is parsed
  correctly.
- **Weixin remote replies.** Slow Weixin remote replies keep their final answer
  in the remote snapshot.

## Known limitations

- **macOS Intel DMG is manual.** The Intel DMG is still built by the on-demand
  workflow after the tag-triggered release creates the GitHub release.

# WispTerm v1.8.0（中文）

> 本版本扩展了 AI chat 与 AI History 工作流，增加了运行中程序的关闭确认，并修复了多项
> Windows 渲染和 agent 执行回归问题。

## 新增

- **AI chat 视觉图片粘贴。** 面向支持视觉的模型，`Ctrl+Shift+V` 现在可以把剪贴板图片
  粘贴到 AI 输入框，并显示附件占位与右上角粘贴提示。
- **AI chat 拖入文件路径。** 把本地文件拖到 chat 输入框时，如果命中当前 chat surface，
  会在光标处插入文件路径。
- **Reasonix AI History 支持。** AI History 现在可以发现并展示 Reasonix 历史记录。
- **面板管理控件。** 右侧面板现在显示关闭按钮，标签页内也可以通过 `Alt` 拖拽交换面板。
- **运行中程序关闭确认。** 关闭标签页、分屏或窗口时，如果终端内仍有运行中的程序，现在会
  先请求确认；新增 `confirm-close-running-program` 配置开关。
- **Agent 终端恢复控制。** Agent 终端执行现在可以发送控制键用于恢复，并会在上一条命令仍
  运行时阻止再次注入命令。

## 修复

- **AI API 请求错误可见。** AI API 请求失败现在会暴露给用户，不再被静默失败路径吞掉。
- **AI chat 图片输入渲染。** 粘贴的图片引用会显示在输入框上方，占位符插入更稳定，审批卡片
  也会随 UI 字号正确缩放。
- **Agent exec 误判失败。** 完成哨兵现在按状态标记匹配，而不是匹配回显命令文本，减少错误
  的超时/失败报告。
- **Windows 渲染异常。** 修复全屏黑屏、多显示器图标残影以及调整大小卡顿。
- **预览路径标点。** 预览路径中的全角标点现在可以正确解析。
- **Weixin 远端回复。** 慢速 Weixin 远端回复现在会把最终答案保留在远端快照中。

## 已知限制

- **macOS Intel DMG 需要手动触发。** Intel DMG 仍会在 tag 触发的 release 创建后，由按需
  工作流单独构建并上传。
